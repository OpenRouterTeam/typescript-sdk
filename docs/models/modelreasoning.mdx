title: "ModelReasoning"

Reasoning effort configuration. Omitted for non-reasoning models and dynamic router models.

## Example Usage

```typescript
import { ModelReasoning } from "@openrouter/sdk/models";

let value: ModelReasoning = {
  mandatory: false,
};
```
