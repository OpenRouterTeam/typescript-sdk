A parameter that accepts one of a discrete set of string values.

## Example Usage

```typescript
import { EnumCapability } from "@openrouter/sdk/models";

let value: EnumCapability = {
  type: "enum",
  values: [
    "1K",
    "2K",
    "4K",
  ],
};
```
