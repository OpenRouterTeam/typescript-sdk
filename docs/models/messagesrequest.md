# MessagesRequest

Request schema for Anthropic Messages API endpoint

## Example Usage

```typescript
import { MessagesRequest } from "@openrouter/sdk/models";

let value: MessagesRequest = {
  messages: [
    {
      content: "Hello, how are you?",
      role: "user",
    },
  ],
  model: "anthropic/claude-4.5-sonnet-20250929",
};
```
