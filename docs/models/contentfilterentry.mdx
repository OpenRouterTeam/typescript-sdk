A custom regex content filter that scans request messages for matching patterns.

## Example Usage

```typescript
import { ContentFilterEntry } from "@openrouter/sdk/models";

let value: ContentFilterEntry = {
  action: "redact",
  pattern: "\\b(sk-[a-zA-Z0-9]{48})\\b",
};
```
