---
title: "ClassifierFiltersValue"
---

Filter value. Use a scalar (string or number) for eq/neq, or an array for in/not_in.


## Supported Types

### `string`

```typescript
const value: string = "Engineering";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `operations.ValueClassifierFilters[]`

```typescript
const value: operations.ValueClassifierFilters[] = [
  "<value>",
];
```
