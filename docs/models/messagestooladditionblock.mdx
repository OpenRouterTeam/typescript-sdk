---
title: "MessagesToolAdditionBlock"
---

Loads a previously deferred tool (declared in `tools` with `defer_loading: true`) mid-conversation without invalidating the prompt cache. Only valid in `role: "system"` messages; requires the `mid-conversation-tool-changes-2026-07-01` beta header.

## Example Usage

```typescript
import { MessagesToolAdditionBlock } from "@openrouter/sdk/models";

let value: MessagesToolAdditionBlock = {
  tool: {
    name: "get_forecast",
    type: "tool_reference",
  },
  type: "tool_addition",
};
```
