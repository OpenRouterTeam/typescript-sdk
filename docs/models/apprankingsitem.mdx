## Example Usage

```typescript
import { AppRankingsItem } from "@openrouter/sdk/models";

let value: AppRankingsItem = {
  appId: 12345,
  appName: "Cline",
  rank: 1,
  totalRequests: 4321,
  totalTokens: "12345678",
};
```
