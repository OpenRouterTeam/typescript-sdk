A timestamped transcript segment, returned when response_format is verbose_json

## Example Usage

```typescript
import { STTSegment } from "@openrouter/sdk/models";

let value: STTSegment = {
  end: 3.2,
  id: 0,
  start: 0,
  text: "Hello there.",
};
```
