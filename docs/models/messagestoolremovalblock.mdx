---
title: "MessagesToolRemovalBlock"
---

Removes a tool from the conversation mid-conversation without invalidating the prompt cache. Only valid in `role: "system"` messages; requires the `mid-conversation-tool-changes-2026-07-01` beta header.

## Example Usage

```typescript
import { MessagesToolRemovalBlock } from "@openrouter/sdk/models";

let value: MessagesToolRemovalBlock = {
  tool: {
    name: "get_weather",
    type: "tool_reference",
  },
  type: "tool_removal",
};
```
