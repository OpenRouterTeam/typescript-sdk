title: "IntentModels"

Per-task-type model overrides layered over the pinned/promoted config for this request only. Overrides apply to text routing; multimodal requests keep the config's vision-capable model tables.

## Example Usage

```typescript
import { IntentModels } from "@openrouter/sdk/models";

let value: IntentModels = {};
```
