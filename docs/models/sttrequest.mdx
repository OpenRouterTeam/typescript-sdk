---
title: "STTRequest"
---

Speech-to-text request input. Accepts a JSON body with input_audio containing base64-encoded audio.

## Example Usage

```typescript
import { STTRequest } from "@openrouter/sdk/models";

let value: STTRequest = {
  inputAudio: {
    data: "UklGRiQA...",
    format: "wav",
  },
  model: "openai/whisper-large-v3",
};
```
