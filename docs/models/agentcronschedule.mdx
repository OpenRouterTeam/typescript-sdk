title: "AgentCronSchedule"
---

Cron schedule. When unset, the agent runs only on explicit invocation.

## Example Usage

```typescript
import { AgentCronSchedule } from "@openrouter/sdk/models";

let value: AgentCronSchedule = {
  expression: "0 9 * * 1-5",
  timezone: "America/Los_Angeles",
};
```
