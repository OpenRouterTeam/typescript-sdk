title: "AgentSource"

Where the agent should run. v1 ships GitHub-only; future versions will add other source types behind the `type` discriminator.

## Example Usage

```typescript
import { AgentSource } from "@openrouter/sdk/models";

let value: AgentSource = {
  repo: "myorg/myapp",
  type: "github",
};
```
