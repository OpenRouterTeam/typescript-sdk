---
title: "CodexModelsResponse"
---

Codex-native model catalog (mirrors the Codex CLI's `ModelsResponse` struct). Served instead of the standard model list when the request is identified as a Codex model-catalog fetch: a `client_version` query parameter plus a Codex `User-Agent` or `Originator` header.
