## Overview

Agent Sessions endpoints

### Available Operations

* [list](#list) - List agent sessions
* [create](#create) - Create an agent session
* [get](#get) - Get an agent session
* [cancel](#cancel) - Cancel an agent session
* [stream](#stream) - Stream session events as SSE

## list

Lists agent sessions visible to the caller, optionally filtered by `agent` and `status`. Listing is backed by the OpenRouter session index and works across providers — but the index itself is not yet implemented, so this endpoint currently returns 501.

### Example Usage

```typescript
import { OpenRouter } from "@openrouter/sdk";

const openRouter = new OpenRouter({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["OPENROUTER_API_KEY"] ?? "",
});

async function run() {
  const result = await openRouter.agentSessions.list();

  for await (const page of result) {
    console.log(page);
  }
}

run();
```

## create

Dispatches a task to a cloud agent. The same endpoint accepts any registered agent slug; the request shape is normalized across providers, with provider-specific behavior available through `extensions` or `passthrough`.

### Example Usage

```typescript
import { OpenRouter } from "@openrouter/sdk";

const openRouter = new OpenRouter({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["OPENROUTER_API_KEY"] ?? "",
});

async function run() {
  const result = await openRouter.agentSessions.create({
    createAgentSessionRequest: {
      agent: "cognition/devin",
      prompt: "Refactor the auth module to use JWT tokens",
      source: {
        branch: "main",
        repo: "myorg/myapp",
        type: "github",
      },
    },
  });

  console.log(result);
}

run();
```
