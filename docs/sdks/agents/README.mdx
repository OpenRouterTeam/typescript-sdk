## Overview

Agents endpoints

### Available Operations

* [list](#list) - List agents
* [create](#create) - Create a user-defined agent
* [delete](#delete) - Delete an agent
* [get](#get) - Get an agent
* [update](#update) - Update an agent
* [sendMessage](#sendmessage) - Talk to an agent to mutate its behavior

## list

Returns the agents the caller can see — owner-scoped by default. Use `owner=public` to also return first-party / public agents.

### Example Usage

```typescript
import { OpenRouter } from "@openrouter/sdk";

const openRouter = new OpenRouter({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["OPENROUTER_API_KEY"] ?? "",
});

async function run() {
  const result = await openRouter.agents.list();

  for await (const page of result) {
    console.log(page);
  }
}

run();
```

## create

Registers a new agent under the calling user / org. Bring-your-own cron schedule, custom code hook, tool list, and natural-language instructions. The agent is dispatched through one of the underlying provider adapters at invoke time. Agents created here show up in the same models / agents pages as first-party agents.

### Example Usage

```typescript
import { OpenRouter } from "@openrouter/sdk";

const openRouter = new OpenRouter({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["OPENROUTER_API_KEY"] ?? "",
});

async function run() {
  const result = await openRouter.agents.create({
    createAgentRequest: {
      adapter: "cognition/devin",
      config: {
        instructions: "Post a standup summary to #eng.",
        schedule: {
          expression: "0 9 * * 1-5",
          timezone: "America/Los_Angeles",
        },
      },
      description: "Posts a standup summary to #eng each weekday at 9am Pacific.",
      displayName: "Daily Standup Bot",
      slug: "myorg/daily-standup",
    },
  });

  console.log(result);
}

run();
```

## sendMessage

Appends a message to the agent's configuration thread. The platform interprets the message and rewrites the agent's instructions, schedule, or tools as appropriate. Use this to evolve an agent over time without manually editing the structured config.

### Example Usage

```typescript
import { OpenRouter } from "@openrouter/sdk";

const openRouter = new OpenRouter({
  httpReferer: "<value>",
  appTitle: "<value>",
  appCategories: "<value>",
  apiKey: process.env["OPENROUTER_API_KEY"] ?? "",
});

async function run() {
  const result = await openRouter.agents.sendMessage({
    agentId: "01h9k2m3-p4q5-r6s7-t8u9-v0w1x2y3z4ab",
    sendAgentMessageRequest: {
      content: "From now on, only run on weekdays.",
    },
  });

  console.log(result);
}

run();
```
